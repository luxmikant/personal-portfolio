# Portfolio Website - Luxmikant

## Overview
An interactive, animated portfolio showcasing backend engineering, cloud-native systems,
AI tools, and Web3 expertise through a storytelling-driven experience with a 3D avatar protagonist.

## Tech Stack
- **Framework**: Next.js 15 + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, React Spring, Motion One, Lenis
- **3D**: Three.js (React Three Fiber)
- **Page Transitions**: Barba.js
- **Deployment**: Vercel

## Key Features
- 3D Avatar with domain-specific transformations & minor idle animations
- Smooth page transitions with Barba.js
- Custom loader UI per domain
- Responsive design (mobile-first)
- Scroll animations with Lenis
- Interactive project showcase
- SEO optimized
